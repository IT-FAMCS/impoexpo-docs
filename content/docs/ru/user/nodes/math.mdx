## Модуль (|x|) [#abs]

<Wrapper>
    ![math abs node](./img/math-abs.png)
</Wrapper>

Берёт число по модулю. Например, `abs(-5) = 5`, `abs(3) = 3`.

<TypeTable type={{
    "ввод": {
        required: true,
        type: "number",
        io: "input",
        default: "0"
    },
    "результат": {
        required: true,
        type: "number",
        description: "Это же число, но по модулю.",
        io: "output"
    }
}}/>
